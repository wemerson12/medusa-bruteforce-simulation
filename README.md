##  Descrição
Este projeto demonstra, de forma **educacional**, como funciona um ataque de **Brute Force de Senhas** utilizando a ferramenta **Medusa** no Kali Linux.  
O objetivo é **apenas acadêmico**, para entender como atacantes podem tentar explorar sistemas e como administradores podem se proteger.

---

##  Ferramentas Utilizadas
- **Kali Linux**
- **Medusa**
- **Serviço alvo simulado** (ex.: SSH ou FTP em ambiente controlado)
